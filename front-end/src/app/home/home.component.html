Movie Quiz

How to play the game.

Movie night quiz is a new quiz game where the player can select the movie genre and is asked seven questions on that genre. The player is award a score for a correct answer and the time is taken to answer the quesition. There is a leaderboard at the end of the game to see all the players currently that have played the quiz.

Points Score

  • Three answers for each question that the user can choose from. One correct, and two incorrect.
  • The timer counts down for 10 seconds.
